Conectar con Firebase

Para que el cuadrante se actualice en tiempo real para todos los participantes, necesitas una base de datos gratuita de Firebase (Google). Sigue estos pasos — son unos 5 minutos:

1. Ve a console.firebase.google.com y crea un proyecto nuevo (nombre: "sorteo-esparto" o el que quieras)
2. Dentro del proyecto, ve a "Realtime Database" → Crear base de datos → Europa → Modo de prueba
3. Ve a Configuración del proyecto (rueda dentada) → pestaña "General" → baja hasta "Tu aplicación" → añade una app web (icono </>) → copia el objeto firebaseConfig

⚙️ Configuración

  • El asistente (wizard) guía la configuración inicial: título, causa, premios, formas de pago.
  • Edita en cualquier momento con ⚙️ Editar configuración.
  • Las fotos de premios se suben con el botón 📷 Subir foto — van a Firebase Storage automáticamente.

🔢 Cuadrante

  • 🟢 Libre · 🟠 Reservado (pendiente pago) · ⚫ Confirmado (pagado).
  • Toca cualquier número para cambiar su estado manualmente.
  • ⬇️ Exportar backup guarda un JSON con todos los datos.
  • 🗑️ Limpiar cuadrante resetea todo (solo para nuevo sorteo).

💵 Pagos

  • Lista todos los confirmados con nombre, teléfono y número.
  • Pago en efectivo: reserva al instante y confirma cuando cobres el euro.
  • Bizum sin concepto: asigna un pago huérfano usando el teléfono del pagador.
  • ⬇️ Exportar CSV descarga todos los participantes en hoja de cálculo.

🎊 Revelar ganadores (tab ONCE)

  • Introduce los 5 dígitos del cupón ONCE del día del sorteo.
  • Sistema calcula: 2 últimas cifras → Premio 1 (derecha) · 2 primeras cifras → Premio 2 (izquierda).
  • Aparece pantalla de celebración con confeti y datos de los ganadores.

  • Realtime Database: reservas, configuración y estados.
