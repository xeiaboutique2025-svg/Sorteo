Conectar con Firebase

Para que el cuadrante se actualice en tiempo real para todos los participantes, necesitas una base de datos gratuita de Firebase (Google). Sigue estos pasos — son unos 5 minutos:

1. Ve a console.firebase.google.com y crea un proyecto nuevo (nombre: "sorteo-esparto" o el que quieras)
2. Dentro del proyecto, ve a "Realtime Database" → Crear base de datos → Europa → Modo de prueba
3. Ve a Configuración del proyecto (rueda dentada) → pestaña "General" → baja hasta "Tu aplicación" → añade una app web (icono </>) → copia el objeto firebaseConfig
